#+title: Models Package
#+property: header-args:common-lisp



Umbrella package for =shiso/models= that re-exports the public API from all
sub-packages.  Users write =(shiso/models:define-model ...)= or use the
short nickname =m= via =:local-nicknames=.

* Package
